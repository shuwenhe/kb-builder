# Sample Documents for KB Builder Demo

This directory should contain `.doc` or `.docx` files that you want to build into a knowledge base.

## Example Usage

1. Place your Word documents here:
   - `docs/document1.docx`
   - `docs/document2.docx`
   - etc.

2. Build the knowledge base:
   ```bash
   python example_build.py
   ```

3. Query the knowledge base:
   ```bash
   python example_load.py
   ```

## Creating Sample Documents

You can create sample .docx files using:
- Microsoft Word
- LibreOffice Writer
- python-docx library

For testing, you can copy any .docx files from your system or create them programmatically.
